## Supported data types:

> <Icon icon="book" iconType="solid" size={15}/>  **LOGS, TRACES, METRICS**

## Configuration

| Name    | Type   | Default                                                                  | Description                                           |
| ------- | ------ | ------------------------------------------------------------------------ | ----------------------------------------------------- |
| Name    | String | Unique identifier within Sawmills                                        |                                                       |
| API Key | String | The Datadog API key to associate yourAgent's data with your organization |                                                       |
| Site    | String | datadoghq.com                                                            | The site of the Datadog intake to send Agent data to. |
